# Coding Dojo | Tulsa: Build Your First Game

### Extra Challenges

* When Pacman hits a ghost, Pacman object disappears and a button to reset the game pops up.
* Add fruit objects randomly to the game board that increase the score by 50 points.
* Have the ghost object "chase" Pacman object. Have it move toward the Pacman one space every time Pacman moves.
